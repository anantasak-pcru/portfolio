ที่สร้างโปรเจกนี้เกิดจากเป็นคนชอบดูซีรี่มากแต่ไม่ค่อยมีเวลาเลยสมัคร VIP แล้วไม่คุ้ม โหลดไว้ดูก็ไม่ได้ที่จริงมันมีเว็บโหลดเเหละแต่ไม่ชอบซับดับชอบซับของ viu บวกกับช่วงนี้ว่างๆ กับกำลังหัดเล่นภาษา PYTHON เลยลองเขียนโปรเเกรมนี้ขึ้นมา 

          
          V1
              - โหลดจาก url เท่านั้น

              - ไม่สามรถโหลด content file เช่น ffmpeg ต้องโหลดมาเอง

          V2
