Android Appclication Giftshop (Java version)

แอพนี้ที่จริงเป็นแแบบฝึกหัดในคลาสเป็นแพเเรกที่เขียนแอนดรอยเลย ทุ่มเทมาก

โจทย์เป็นแอพของฝาก ไม่รู้ว่าจะเอาข้อมูลของฝากมาจากไหนเลยทำมันคล้ายๆตลาดให้ทุกคนมาลงขายได้ซ่ะเลย 

          
            ทำอไรได้บ้าง
                - เพิ่ม ลบ แก้ไข สินค้าสำหรับสมาชิก
                - ดูสินค้าสำหรับบุคคลทั่ว
                - มีระบบสมาชิก
            
            โปรเจกนี้ใช้
                - ภาษา Java
                - api ทั้งหมด ใช้ firebase
